# Find max number between A and B

## Logic/ Pseudocode

Algorithm MaxNumber(A, B)

Input: Integer A, B
Output: Integer Max

1. Start
2. if A > B then
    return A
3. Else
    return B
4. End

## Dry Run

Example 01: A = 10; B = 15
10 > 15 --> False --> return 15

Example 02: A = 6; B = 6
6 > 6 --> False --> return 6

Example 03: A = 5; B = 2
5 > 2 --> True --> return 5

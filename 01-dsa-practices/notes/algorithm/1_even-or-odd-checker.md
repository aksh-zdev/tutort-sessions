# Even or Odd Checker

## Logic/ Pseudocode

    Algorithm IsEvenOrOdd(n)

    Input: Integer n
    Output: "Even" or "Odd"

    1. Start

    2. If n % 2 == 0 then
        Return "Even"

    3. Else
      Return "Odd"

    5. End

## Dry Run

    Example 01: n = 16
      16 % 2 --> 0 Condition true --> return "Even"

    Example 02: n = 99
      99 % 2 --> not 0 Condition false --> return "Odd"
